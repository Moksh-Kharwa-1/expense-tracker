# Expense Tracker

A command-line personal expense tracker built with Python.

## Features
- Add expenses with date, payment method, recipient, and amount
- Input validation for all fields — date format, payment method, positive amounts
- Groups expenses by category automatically
- Shows spending summary with category totals
- Overspend warning when expenses exceed income
- Cancel any entry mid-way without losing previous data

## Built With
- Python
- datetime library

## How to Run
1. Clone the repository
2. Run expense_tracker.py in any Python environment
3. Follow the prompts to add expenses
4. Type 'done' to finish and see your summary
5. Type 'cancel' at any field to scrap the current entry
